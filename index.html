User API Documentation

Welcome to the User API documentation. This API provides endpoints for user management including registration, authentication, and search functionality.

Register User

POST /api/users/register

Register a new user with username, email, and password.

Login User

POST /api/users/login

Authenticate user with email and password.

Search Users

GET /api/users/search

Search for users using query parameters.

API Version 1.0 | Running on Port 3000
